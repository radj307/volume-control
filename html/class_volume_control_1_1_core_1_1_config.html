Property Documentation

◆ AdvancedHotkeys

bool VolumeControl.Core.Config.AdvancedHotkeys = false
getset

Gets or sets the hotkey editor mode, which can be either false (basic mode) or true (advanced mode).

Advanced mode allows the user to perform additional actions in the hotkey editor:

  • Create and delete hotkeys.
  • Rename hotkeys.
  • View and modify the action bindings of hotkeys.

Default: false

◆ AllowTransparency

bool VolumeControl.Core.Config.AllowTransparency = true
getset

Gets or sets whether the main window allows transparency effects.

  • When true, window background transparency is allowed.
  • When false, window background transparency isn't allowed;
    only controls within the window may use transparent background effects.
    (See System.Windows.Controls.Control.Background)

Default: true
