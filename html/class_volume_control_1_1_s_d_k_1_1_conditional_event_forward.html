A delegate method or lambda that determines whether or not events received through the Handler(object?, object) are forwarded to the Event.

Returns
true causes the event to be forwarded via the Event event; false causes the event to be ignored, and Event is not triggered.

object  e

An event handler that can be attached to any EventHandler or EventHandler<TEventArgs> compatible event trigger.
Note that if the generic TEventArgs type of the event handler is not an object type, such as a tuple, you will have to 'wrap' the handler call with a lambda:

ConditionalEventForward cef = new(() => allowEventCondition); MyEvent += (s, e) => cef.Handler(s, e);

Parameters
senderThis is forwarded to the first parameter in any handlers attached to Event.
eThis is forwarded to the second parameter in any handlers attached to Event.
