Detailed Description

Contains methods related to audio devices, sessions, and the underlying selection mechanics responsible for providing context to actions.
You can use this object to manipulate any audio session on the system.

◆ VolumeChanged

Triggered when the volume level or mute state of SelectedDevice or SelectedSession is changed.
To differentiate between devices and sessions, see the VolumeEventArgs.Target property.

Note that this event is only triggered by AudioAPI methods, and as a result does not trigger when the volume is changed using the UI or by any other means!
A non-exhaustive list of methods that trigger this event:

  • IncrementDeviceVolume(int)
  • DecrementDeviceVolume(int)
  • SetDeviceMute(bool)
  • ToggleDeviceMute()
  • IncrementSessionVolume(int)
  • DecrementSessionVolume(int)
  • SetDeviceMute(bool)
  • ToggleDeviceMute()
