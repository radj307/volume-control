Release types, for use with version self-identification CI scripts with the ReleaseType assembly attribute.

Enumerator
NONE 

Null type / Error Type.

TESTING 

Testing build, not meant for release.

PRERELEASE 

Pre-release (unfinished) version.

CANDIDATE 

Release candidate version. These are almost ready to be released, but haven't been extensively tested yet.

NORMAL 

A normal release. This does not include revisions!

REVISION 

This is a slight modification to an existing release that doesn't qualify as a whole new version.
