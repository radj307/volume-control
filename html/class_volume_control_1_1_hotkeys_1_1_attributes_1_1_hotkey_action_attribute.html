Detailed Description

Indicates that a method is a valid hotkey action.
If you don't specify a name parameter, it is parsed from the name of the method it is attached to.

To create a hotkey action, attach this attribute to the owning method.

string VolumeControl.Hotkeys.Attributes.HotkeyActionAttribute.ActionName => _actionName # (InterpolateName ? ParseActionName(_originalName) : _originalName)

The name shown in the GUI when selecting a hotkey action.
This cannot be changed directly post-construction time, but may be influenced by InterpolateName after construction.
If this is set to an empty string, the action is always ignored and will never be selectable.

Note that this is also the name used when saving hotkeys to the config file.

◆ InterpolateName

bool VolumeControl.Hotkeys.Attributes.HotkeyActionAttribute.InterpolateName
getset

Gets or sets whether the action name is interpolated by inserting spaces between each of the uppercase letters to split the name by words.
If your action is named using abbreviations or you don't want this behaviour to occur, set this to false.

Note that when this property changes, the ActionName property is invalidated causing it to be regenerated the next time it is requested.
