Detailed Description

Indicates that a method is a valid hotkey action.
If you don't specify a name parameter, it is parsed from the name of the method it is attached to.

To create a hotkey action, attach this attribute to the owning method.

VolumeControl.Hotkeys.Attributes.HotkeyActionAttribute.HotkeyActionAttribute ( [CallerMemberName] string  name = "")

Use this attribute to mark the associated method as a hotkey action.

Parameters
nameThis determines the value of the Name property, which is used in conjunction with the InsertSpacesInName & GroupName properties to determine the action name shown in the GUI.

When this parameter is omitted, the action name is automatically set to the method name using reflection.
Example: Consider a method named "SessionVolumeUp", for which the default value of name is either "SessionVolumeUp" when InsertSpacesInName is false or "Session Volume Up" when InsertSpacesInName is true.
