enum VolumeControl.Helpers.HotkeyActionManager.NameConflictResolution
strongprotected

Determines how duplicated action names are handled by the parser.

Enumerator
Throw 

When an action with a conflicting name is found, an exception is thrown that includes the IActionBinding.Name of the duplicate action.

Ignore 

When an action with a conflicting name is found, it is ignored.

Overwrite 

When an action with a conflicting name is found, it will overwrite the action already in the Bindings list.
