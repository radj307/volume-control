◆ AdvancedHotkeyMode

bool VolumeControl.Core.Helpers.VCSettingsContainer.AdvancedHotkeyMode
getset

Gets or sets the hotkey editor mode, which can be either false (basic mode) or true (advanced mode).

Advanced mode allows the user to perform additional actions in the hotkey editor:

  • Create and delete hotkeys.
  • Change the action bindings of hotkeys.
  • Rename hotkeys.
