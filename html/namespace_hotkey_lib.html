enum HotkeyLib.HotkeyRegistrationState : byte
strong

Enum containing the various states that a hotkey registration may be in.

Enumerator
UNREGISTERED 

Hotkey is not registered.

REGISTERED 

Hotkey is registered.

FAILED 

The previous attempt to register the hotkey failed, and it is now in a failed state.
This can usually be cleared by simply re-registering it.

◆ Modifier

enum HotkeyLib.Modifier : uint
strong

Enumerator that represents hotkey modifiers.

These correspond to the fsModifiers parameter of the HotkeyAPI.RegisterHotkey(IntPtr, int, uint, System.Windows.Forms.Keys) function.

Enumerator
NONE 

No Modifier Keys.

ALT 

Either ALT key must be held down.

CTRL 

Either CTRL key must be held down.

SHIFT 

Either SHIFT key must be held down.

WIN 

Either WINDOWS key was held down. These keys are labeled with the Windows logo. Keyboard shortcuts that involve the WINDOWS key are reserved for use by the operating system.

NOREPEAT 

Changes the hotkey behavior so that the keyboard auto-repeat does not yield multiple hotkey notifications. Windows Vista: This flag is not supported.
