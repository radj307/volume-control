Detailed Description

AudioSession management container type.
This is implemented as a viewmodel in the AudioAPI class.

This object does not maintain ownership of AudioSession objects; rather, it utilizes AudioDevice.Sessions to create a single binding point for the UI.

This object cannot be instantiated externally.
