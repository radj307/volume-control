◆ LockSelection

bool VolumeControl.SDK.IListDisplayTarget.LockSelection
getset

Gets or sets whether the user can change the SelectedItem property of the ListView control by clicking on an item in the list.

While this is used by the Audio Sessions & Audio Devices display targets, this property is not the same thing; this controls whether the actual ListView control allows the user to change the selection; it does not work the other way around!
