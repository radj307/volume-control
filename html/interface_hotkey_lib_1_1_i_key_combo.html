Detailed Description

Represents a combination of one keyboard key and any number of modifier keys.

Property Documentation

◆ Alt

bool HotkeyLib.IKeyCombo.Alt
getset

Get or set whether the Alt modifier key is enabled. This corresponds to the first bit in the modifier key bitfield. Both the left and right modifier keys apply here, there is no method available to limit the key combination to only the left or right modifier key.
