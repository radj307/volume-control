◆ HotkeyActionAttribute()

VolumeControl.Core.Attributes.HotkeyActionAttribute.HotkeyActionAttribute ( [CallerMemberName] string  name = "")

Indicates that the attached method defines a hotkey action.

Parameters
nameThis determines the value of the Name property, which is used in conjunction with the InsertSpacesInName & GroupName properties to determine the action name shown in the GUI.

When this parameter is omitted, the action name is automatically set to the method name using reflection.
Example: Consider a method named "SessionVolumeUp", for which the default value of name is either "SessionVolumeUp" when InsertSpacesInName is false or "Session Volume Up" when InsertSpacesInName is true.

◆ SeparateWordsInString()

static string VolumeControl.Core.Attributes.HotkeyActionAttribute.SeparateWordsInString ( string  str)
static

Separates str by inserting a space before each capitalized letter that appears directly after at least one non-whitespace character.

This works by replacing occurrences of the regular expression "<see langword="\B([A-Z])"/>" with the replacement expression "<see langword=" $1"/>".

Parameters
strA string to separate.
Returns
The given string with a single space character inserted between each word.

◆ InsertSpacesInName

bool VolumeControl.Core.Attributes.HotkeyActionAttribute.InsertSpacesInName = true
getset

Gets or sets whether the Name property will be formatted before being used.

When set to true, the Name property is passed through the SeparateWordsInString(string) function to insert spaces between capitalized letters; otherwise when false, the Name property is not modified and is used as-is.
