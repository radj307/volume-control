Detailed Description

A static class that provides methods for extracting icons from files, directories, or DLLs, as well as directly loading them from icon or image files.

string VolumeControl.WPF.IconExtractor.path
static

Parses the given icon path into a usable path string with an optional icon index number.

Parameters
iconPathSpecifies the imageUri and index number of the target icon.
Returns
The resolved path as a string, and the index of the icon if one was specified in the iconPath . If no index was specified, the returned index is -1.
