Detailed Description

A static class that provides methods for extracting icons from files, directories, or DLLs, as well as directly loading them from icon or image files.

Parses the specified iconPath by separating the filepath and the icon index.

This is used by ExtractFromPath(string, bool) to parse icon paths that point to a specific icon within a DLL that may contain multiple icons.

Parameters
