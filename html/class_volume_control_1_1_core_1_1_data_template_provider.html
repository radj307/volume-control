Detailed Description

Provides a DataTemplate for some hardcoded type.
The Type of a derived class can be specified for the HotkeyActionSettingAttribute.DataTemplateProviderType property to specify which DataTemplate will be used to display an editor control for that action setting.

To create a DataTemplateProvider, first create a new class that implements it:

public class IntDataTemplateProvider : DataTemplateProvider
{
public override DataTemplate ProvideDataTemplate()
{
// Create a TextBox (via a factory) and bind its Text property to the action setting's Value property:
var textBoxFactory = new FrameworkElementFactory(typeof(TextBox));
textBoxFactory.SetBinding(TextBox.TextProperty, new Binding("Value"));
return new DataTemplate(typeof(int)) { VisualTree = textBoxFactory };
}
}

abstract DataTemplate VolumeControl.Core.DataTemplateProvider.ProvideDataTemplate ( )
pure virtual

Gets the DataTemplate instance to use for displaying an editor control for the action setting's value.

The returned DataTemplate will always be used with a DataContext of the Input.Actions.Settings.IActionSettingInstance instance for that action setting. Therefore, valid data binding paths can be relative to an Input.Actions.Settings.IActionSettingInstance instance, or absolute by directly specifying a binding Source.
To bind to the value of the action setting, set the data binding's path to "Value".

Returns
A new DataTemplate instance specifying the controls to use for displaying an action setting value editor.
