Detailed Description

Represents an audio device endpoint.

This object uses the MMDevice API from NAudio, and implements the following interfaces:

  • IDevice
  • INotifyPropertyChanged
  • IDisposable

Some properties in this object require the NAudio library.
If you're writing an addon, install the 'NAudio' nuget package if you need to be able to use them.

◆ Icon

ImageSource? VolumeControl.Audio.AudioDevice.Icon => SmallIcon ?? LargeIcon

Returns SmallIcon or if it is set to null, LargeIcon is returned instead.
If both icons are set to null, this returns null.

Note that the icon properties all use internal caching; you don't need to worry about using these properties repeatedly for fear of performance loss, as the only time the icons are actually retrieved is the first time any of the icon properties are called.

◆ IconPath

string VolumeControl.Audio.AudioDevice.IconPath => _device.IconPath

This is the location of this device's icon on the system and may or may not actually be set.

This can point to nothing, one icon in a DLL package, an executable, or all sorts of other formats.
Do not use this for retrieving icon images directly, instead use SmallIcon/LargeIcon.
You can also use Icon directly if you don't care about the icon size.
