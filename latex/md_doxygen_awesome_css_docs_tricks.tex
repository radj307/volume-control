\hypertarget{md_doxygen_awesome_css_docs_tricks_autotoc_md14}{}\doxysection{Class Diagrams with Graphviz}\label{md_doxygen_awesome_css_docs_tricks_autotoc_md14}
To get the best looking class diagrams for your documentation, generate them with Graphviz as vector graphics with transparent background\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\# Doxyfile}
\DoxyCodeLine{HAVE\_DOT = YES}
\DoxyCodeLine{DOT\_IMAGE\_FORMAT = svg}
\DoxyCodeLine{DOT\_TRANSPARENT = YES}

\end{DoxyCode}
\hypertarget{md_doxygen_awesome_css_docs_tricks_autotoc_md15}{}\doxysection{Disable Dark Mode}\label{md_doxygen_awesome_css_docs_tricks_autotoc_md15}
If for some reason you don\textquotesingle{}t want the theme to automatically switch to dark mode depending on the browser preference, you can disable dark mode by adding the {\ttfamily light-\/mode} class to the html-\/tag in the header template\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{<html xmlns="{}http://www.w3.org/1999/xhtml"{} class="{}light-\/mode"{}>}

\end{DoxyCode}


The same can be done to always enable dark-\/mode\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{<html xmlns="{}http://www.w3.org/1999/xhtml"{} class="{}dark-\/mode"{}>}

\end{DoxyCode}


{\bfseries{This only works if you don\textquotesingle{}t use the dark-\/mode toggle extension.}}\hypertarget{md_doxygen_awesome_css_docs_tricks_autotoc_md16}{}\doxysection{Choosing Sidebar Width}\label{md_doxygen_awesome_css_docs_tricks_autotoc_md16}
If you have enabled the sidebar-\/only theme variant, make sure to carefully choose a proper width for your sidebar. It should be wide enough to hold the icon, project title and version number. If the content is too wide, it will be cut off.


\begin{DoxyCode}{0}
\DoxyCodeLine{html \{}
\DoxyCodeLine{    /* Make sure sidebar is wide enough to contain the page title (logo + title + version) */}
\DoxyCodeLine{    -\/-\/side-\/nav-\/fixed-\/width: 335px;}
\DoxyCodeLine{\}}

\end{DoxyCode}


The choosen width should also be set in the Doxyfile\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\# Doxyfile}
\DoxyCodeLine{TREEVIEW\_WIDTH = 335}

\end{DoxyCode}
