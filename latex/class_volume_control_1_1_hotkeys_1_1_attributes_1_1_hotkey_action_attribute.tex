\doxysubsection{Detailed Description}
Indicates that a method is a valid hotkey action.~\newline
If you don\textquotesingle{}t specify a name parameter, it is parsed from the name of the method it is attached to. 

To create a hotkey action, attach this attribute to the owning method.
