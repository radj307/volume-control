\doxysubsection{Detailed Description}
Represents an audio device endpoint. 

This object uses the MMDevice \mbox{\hyperlink{namespace_volume_control_1_1_a_p_i}{API}} from NAudio, and implements the following interfaces\+:
\begin{DoxyItemize}
\item IDevice 
\item INotify\+Property\+Changed 
\item IDisposable 
\end{DoxyItemize}Some properties in this object require the NAudio library.~\newline
