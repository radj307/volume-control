Simple, universal application-\/specific volume hotkeys \& more. ~\newline
 Designed for adjusting the volume of your music independently of other programs, such as games and Vo\+IP. ~\newline
\hypertarget{index_autotoc_md0}{}\doxysubsubsection{What it does}\label{index_autotoc_md0}

\begin{DoxyItemize}
\item Adds completely user-\/configurable global hotkeys to Windows.
\begin{DoxyItemize}
\item You can add, remove, and rename hotkeys, as well as configure the \textquotesingle{}action\textquotesingle{} that they perform. ~\newline
 This requires enabling $\ast$\textquotesingle{}Advanced Hotkey Mode\textquotesingle{}$\ast$, which is a fancy way of saying $\ast$\textquotesingle{}Check the box labelled with \textquotesingle{}{\bfseries{Advanced Hotkeys}}\textquotesingle{}, which is located in the {\bfseries{Settings}} tab\textquotesingle{}$\ast$
\item The hotkey actions mentioned earlier are fairly diverse, and can be expanded in the form of \href{https://github.com/radj307/volume-control/wiki}{\texttt{ user-\/created plugins}}.
\end{DoxyItemize}
\item Integrates a functionally superior alternative to the Windows Audio Mixer. ~\newline

\begin{DoxyItemize}
\item Includes volume slider trackbars and textboxes so you can specify a volume level with the keyboard.
\begin{DoxyItemize}
\item Like the Windows Audio Mixer, you can do this with any running process independently at any time.
\end{DoxyItemize}
\item Allows you to specify the \textquotesingle{}target\textquotesingle{} application using the mixer, in addition to the target switching hotkeys.
\end{DoxyItemize}

First, enable the {\bfseries{Volume Up}} \& {\bfseries{Volume Down}} hotkeys from the {\bfseries{Hotkeys}} tab by checking the box to the left of the hotkey name. If you don\textquotesingle{}t have a volume slider, change the key from the dropdown.

{\bfseries{NOTE\+:}} Hotkeys cannot be enabled if their associated key is set to {\ttfamily None}.

Next, let\textquotesingle{}s set a target application to test the hotkeys with. ~\newline
 Start playing some audio from any application, then return to the {\bfseries{Mixer}} tab, click {\bfseries{Reload}}, then click the {\bfseries{Select}} button next to the test application, and try using the volume hotkeys. ~\newline
