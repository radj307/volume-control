{\bfseries{Doxygen Awesome}} is a custom {\bfseries{CSS theme for Doxygen HTML-\/documentation}} with lots of customization parameters.\hypertarget{md_doxygen_awesome_css__r_e_a_d_m_e_autotoc_md24}{}\doxysection{Motivation}\label{md_doxygen_awesome_css__r_e_a_d_m_e_autotoc_md24}
I really like how the Doxygen HTML-\/documentation is structured! But IMHO it looks a bit outdated.

This theme is an attempt to update the visuals of Doxygen without changing its overall layout too much.\hypertarget{md_doxygen_awesome_css__r_e_a_d_m_e_autotoc_md25}{}\doxysection{Features}\label{md_doxygen_awesome_css__r_e_a_d_m_e_autotoc_md25}

Copy the file {\ttfamily doxygen-\/awesome.\+css} from this repository into your project or add this repository as submodule and check out the latest release\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{git submodule add https://github.com/jothepro/doxygen-\/awesome-\/css.git}
\DoxyCodeLine{cd doxygen-\/awesome-\/css}
\DoxyCodeLine{git checkout v2.0.3}

\end{DoxyCode}


Choose one of the theme variants and configure Doxygen accordingly\+:








\begin{DoxyEnumerate}
\item {\bfseries{Base theme}}\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\# Doxyfile}
\DoxyCodeLine{GENERATE\_TREEVIEW      = YES \# optional. Also works without treeview}
\DoxyCodeLine{HTML\_EXTRA\_STYLESHEET  = doxygen-\/awesome-\/css/doxygen-\/awesome.css}

\end{DoxyCode}

\item {\bfseries{Sidebar-\/only theme}}\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\# Doxyfile}
\DoxyCodeLine{GENERATE\_TREEVIEW      = YES \# required!}
\DoxyCodeLine{HTML\_EXTRA\_STYLESHEET  = doxygen-\/awesome-\/css/doxygen-\/awesome.css \(\backslash\)}
\DoxyCodeLine{                         doxygen-\/awesome-\/css/doxygen-\/awesome-\/sidebar-\/only.css}

\end{DoxyCode}

\end{DoxyEnumerate}
