\doxysubsection{Detailed Description}
Helper object for C\# Class Libraries being unable to get the current \mbox{\hyperlink{namespace_volume_control_1_1_w_p_f}{WPF}} window handle. 

This can be used to receive window messages for the current window even from within non-\/\+WPF projects.
